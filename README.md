# ExploreTravelore-TravelBlog

ExploreTravelore-TravelBlog is a travel blog application that allows users to view, edit, add, update, and delete travel blog posts.

## Tech stack
- **Next JS**
- **MUI**
- **Redux Toolkit** 
- **React Toastify**
